# rkt rundeck

 
## overview

This project use rkt to create a rundeck application :
* postgresql database
* 1 to n rundeck server
* 1 haproxy

-------- 

## Requirements

* rkt
* acbuild

## Configuration

edit env file

## Build project

* make build


## Start application

* make start

 
## Stop application

* make stop


## Clean containers

* make clean


## Clean datas

* make clean_data
